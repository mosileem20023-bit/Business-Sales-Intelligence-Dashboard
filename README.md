# Sales Performance Analytics Dashboard (Power BI)

This project presents a multi-page Power BI dashboard designed to analyze sales performance, customer behavior, product insights, and profitability using interactive visuals and DAX measures.

## Dashboard Pages
- **Sales Overview Analysis**
- **Customer Analysis**
- **Product Analysis**
- **Profit Analysis**

## Key Features
- Star schema data model
- Custom DAX measures for KPIs and profitability
- Interactive slicers and drill-down analysis
- Sales, customer, product, and profit insights
